Detailed Description

waveファイルを扱うためのクラス。

waveファイルの同期読み込み、非同期読み込みをサポートします。 CWaveFile::Readは同期読み込み、CWaveFile::ReadAsyncは非同期読み込みが行われます。 ReadAsyncを使用した場合はCWaveFile::IsReadEndを使用して同期をとって下さい。
