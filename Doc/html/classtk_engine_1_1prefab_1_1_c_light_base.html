bool tkEngine::prefab::CLightBase::Start ( )
finaloverridevirtual

Updateの直前で呼ばれる開始処理。

本関数がtrueを返すとゲームオブジェクトの準備が完了したと判断されて Update関数が呼ばれ出します。trueを返して以降はStart関数は呼ばれなくなります。 ゲームオブジェクトの初期化に複数フレームかかる場合などはfalseを返して、初期化ステップなどを使って 適切に初期化を行ってください。
