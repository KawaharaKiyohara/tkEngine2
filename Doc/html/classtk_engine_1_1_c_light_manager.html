Detailed Description

ライトの管理者

インスタンス化されたCLightBaseのインスタンスはすべてCLightManagerに登録されます。 このクラスのインスタンスはCGraphicsEngineクラスのみが保持しており、その他のクラスでインスタンスを生成することはできません。
