Detailed Description

ストップウォッチクラス。

C::のStopwatchのように使える。

サンプルコード
Stopwatch sw;
sw.Start(); //計測開始。
for(int i = 0; i < 100; i++ ){
}
sw.Stop(); //計測終了
printf("経過時間 = %lf(単位：秒)\n", sw.GetElapsed());
printf("経過時間 = %lf(単位：ミリ)\n", sw.GetElapsedMillisecond());
