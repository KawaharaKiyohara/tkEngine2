◆ CalcScreenPositionFromWorldPosition()

void tkEngine::CCamera::CalcScreenPositionFromWorldPosition ( CVector2 &  screenPos,
const CVector3 &  worldPos 
) const

ワールド座標からスクリーン座標を計算する。

計算されるスクリーン座標は画面の中心を{0,0}、左上を{画面の幅*-0.5,画面の高さ*-0.5} 右下を{画面の幅*0.5,画面の高さ*0.5}とする座標です。

Parameters
[out]screenPosスクリーン座標の格納先。
[in]worldPosワールド座標。
